Introduction

This is a really simple lightbox jquery plugin. It is intended for static websites – it is not a CMS plugin (e.g. it is not for WordPress).

This minified jquery plugin (js/lightbox.min.js) is 3KB and the accompanying minified stylesheet (css/lightbox.min.css) is 1KB.

Also required is an icon font for the lightbox:

  1. the dismiss icon,
  2. the group icon,
  3. the group next icon,
  4. the loading icon,

For these I have included Font Awesome (which is downloaded from http://www.bootstrapcdn.com/#fontawesome_tab). If you are using different icon font already in your project, you can find the "fa" classes within <i> tags within the jquery plugin file and swap as appropriate for your preferred icon font.

Setting up the Really Simple Lightbox in your project

Follow these three easy steps to get started:

  1. The simplest way to include associate the required jquery files and stylesheet with your project is to copy these three lines into your webpage's <head> : 
    <link rel="stylesheet" href="css/lightbox.min.css">
<script src="http://code.jquery.com/jquery-1.11.0.min.js"></script>
<script src="js/lightbox.min.js"></script>
    If you are unsure of how to do this, look at how I have done it in the source code of this current page, and do the same in your webpage.

    Note that the second line in the example code above, the required jQuery library is being called from the Google Hosted Libraries repository.

  2. Then, copy accross to your project these three folders: css/, js/ and fonts/. (If you do not plan to edit my unminified lightbox.js or lightbox.css you can delete them.)

  3. And then, here is all you need to do to create an instance of a lightbox on your webpage:

    1. For a link: 
      <a href="images/filename.jpg" rel="lightbox">Link</a>
      Click me
      — A http://placekitten.com image should appear in a lightbox.
    2. For a thumbnail: 
      <a href="images/filename.jpg" rel="lightbox"><img src="images/filename-thumb.jpg"></a>
      — You should be able to see a http://placekitten.com image to the left. Click / tap the 'thumbnail' image and the same image (enlarged) should appear in the lightbox.

That's all you need – for single images.

But please read on, to see how you can add a lightbox title, and how you can set up image groups, such as you may need in image galleries.

How the Really Simple Lightbox works

You can have taller-than-wide images (e.g. portraits) or wider-than-tall images (e.g. landscapes), or square images. The lightbox jQuery script will figure out how to display the image by detecting the image's actual width and height, and by detecting the visitor's window width and height, and then making the necessary adjustments so that the image appears neatly on the screen.

To dismiss a lightbox, click the "x" symbol that will appear in the lightbox top-right corner. Or else click anywhere on the darkened page. And if the image fails to appear, click the darkened background or click the "lightbox loading" spinner.

In the CSS I have styled any lightbox thumbmail image (i.e. any image wrapped in an <a> containing a rel="___") to have a white padding (frame) and a thin grey box-shadow. And when a visitor hovers over the thumbnail image, they will see that the shadow turns blue and gently increases in 'depth' (this effect fades out too when you exit the image). This blue color indicates to the visitor that he/she can click on the thumbnail image.

Lightbox titles

If you require a title to appear above the lightbox image, simply add title="Your image title" into the <a> tag, as follows:

<a href="images/filename.jpg" rel="lightbox" title="Something">
    <img src="images/filename-thumb.jpg">
</a>

The title can be a single word, or anything up to a short sentance (e.g. 10 words). Try it, to see what you can fit in. The wider the image, the more words you can have in the title.

Image Groups

But if you have a group of images that need to be viewed in a combined lightbox, then all you need to do is add rel="lightbox-group-XX to the <a> of each image in the group.

For example:

    <a href="images/filenameA.jpg" rel="lightbox-group-1" title="Whatever">
        <img src="images/filenameA-thumb.jpg">
    </a>
    <a href="images/filenameB.jpg" rel="lightbox-group-1" title="Whatever">
        <img src="images/filenameB-thumb.jpg">
    </a>
    <a href="images/filenameC.jpg" rel="lightbox-group-1" title="Whatever">
        <img src="images/filenameC-thumb.jpg">
    </a>

Just be sure to include rel="lightbox-group-something" in the <a> tag. ("something" can be a number to identify your image-group, or a letter, or a word.)

Click on these images below, and you will see that they are linked together in their lightboxes, with a back-arrow and forward-arrow. The group will 'cycle' if you continue clicking one arrow, starting at the beginning again (or starting at the end again, if the visitor is cycling backwards). There is no 'timed automatic' cycle, so your visitors can spend as long as they like looking at your images.

Really Simple Grid system

If you are using this jquery lightbox plugin in conjunction with a CSS grid system such as 960.gs or a framework such as Bootstrap or Foundation, then you can use their grid system for laying out an image gallery. But if you do not already have a grid system in yyour project, I offer you my "really simple grid system."

  1. First, you will need to copy my grid system's styles from base.css (I have put them here, because they are not a necessary part of the lightbox.)
  2. Then, all you need to do to use it is create an outer <div> and give it a class="grid", and then put 2, 3 or 4 <div>'s inside, giving them classes with names such as "half", "third" or "quarter".
<div class="grid">
    <div class="third">Your first third column content</div>
    <div class="third">Your middle third column content</div>
    <div class="third">Your last third column content</div>
</div>

For example:

.half
.half
.third
.third
.third
.quarter
.quarter
.quarter
.quarter
.third
.two-thirds
.quarter
.three-quarters

(So that you san see the examples above, I have wrapped all in another <div>, given it class="example", and used this class name to assign some styles to the <div>'s inside. You don't need to do this yourself.)

How the Really Simple Grid system responds to different window widths

For small screenes (phones): the grid 'column' blocks are stacked certically.

For screen widths from 40rem (640px) and up: the columns are arranged horizontally (using float:left) with widths of, e.g., half, third or quarter of the available space.

For screen widths 80rem (1280px) and up: the "grid" class maximum width is 70rem (1200px).

This really simple grid system is not a necessary part of the lightbox jquery plugin, so I have included it's styles in the example webpage stylesheet (base.css).

Note also that as an integral part of the grid system I have set all <img> images within the .grid wrapper to have a maximum width of 100% their containing column. This means that where an image width is greater than the width of a column (whether quarter, third, half or full content width), it will be constrained to fit its containing <div>. Thus, using thumbnail images that are relatively large (e.g. 250px width or greater, if you are using "quarter" columns) will make them fill the available width.
