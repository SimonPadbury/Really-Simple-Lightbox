Introduction

This is a really simple lightbox jquery plugin for static websites.

The minified jquery plugin (js/lightbox.min.js) is 3KB and the accompanying minified stylesheet (css/lightbox.min.css) is 1KB.

Also required is an icon font for the lightbox:

  1. dismiss icon,
  2. group previous icon,
  3. group next icon,
  4. CSS3-animated loading icon,

For these I have included Font Awesome. (Swap these out of you're using a different icon font.)

Lightbox Setup

Step 1: Include the lightbox stylesheet and JavaScript, and jQuery

The simplest way to include associate the required jquery files and stylesheet with your project is to copy these three lines into your webpage's <head>:

<-- Stylesheet -->
<link rel="stylesheet" href="css/lightbox.min.css">

<-- Scripts -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
<script src="js/lightbox.min.js"></script>

Then, copy lightbox.min.js and lightbox.min.css into your project.

 

Step 2: Create an instance of the lightbox on your webpage

Here is all you need to do to create an instance of a lightbox on your webpage.

For a link:

<a href="images/filename.jpg" data-lightbox="lightbox">Link</a>
Click me

— A placeholder image should appear in a lightbox.

For a thumbnail:

<a href="images/filename.jpg" data-lightbox="lightbox">
  <img class="lightbox-thumbnail" src="images/filename.jpg">
</a>

You may use a smaller (thumbnail) image within the link. (But here I have used the same image, and relied upon img { max-width: 100%, height: auto; } to control the size of the 'thumbnail'.)

— You should be able to see a placeholder image to the left. Click the 'thumbnail' image and the same image (enlarged) should appear in the lightbox.

That's all you need – for single images. But please read on, to see how you can add a lightbox title, and how you can set up image groups, such as you may need in image galleries.

How Really Simple Lightbox works

The lightbox script takes the href="" linked image, wraps it in a <div> and <img> tag, and presents it as a popover lightbox. The script also adds the title, dismiss symbol, and also previous/next navigation links if the lightbox is in a group, a.k.a. gallery. (More information will be given below in the relevant sections.)

You can have taller-than-wide images (e.g. portraits) or wider-than-tall images (e.g. landscapes), or square images. The lightbox script will figure out how to display the image by detecting the image's actual width and height, and by detecting the visitor's window width and height, and then making the necessary adjustments so that the image appears neatly on the screen.

To dismiss a lightbox, click the symbol that will appear in the top-right corner. Or else click anywhere on the darkened page. And if the image fails to appear, click the darkened background or click the "lightbox loading" spinner.

In the CSS I have styled any lightbox thumbmail image (i.e. any image wrapped in an <a> containing a rel="____") to have a white padding (frame) and a thin grey box-shadow. And when a visitor hovers over the thumbnail image, they will see that the shadow turns blue and gently increases in 'depth' (this effect fades out too when you exit the image). This blue color indicates to the visitor that he/she can click on the thumbnail image.

Lightbox titles

If you require a title to appear above the lightbox image, simply add title="Your image title" into the <a> tag, as follows:

<a href="images/filename.jpg" data-lightbox="lightbox" title="Something">
    <img class="lightbox-thumbnail" src="images/filename-thumb.jpg">
</a>

The title should be short, because it needs to fit on a small-screen (phones) lightbox.

Image Groups

But if you have a group of images that need to be viewed in a combined lightbox, then all you need to do is add data-lightbox="lightbox-group-XX to the <a> of each image in the group.

For example:

<a href="images/filenameA.jpg" data-lightbox="lightbox-group-1" title="Whatever">
    <img class="lightbox-thumbnail" src="images/filenameA-thumb.jpg">
</a>

<a href="images/filenameB.jpg" data-lightbox="lightbox-group-1" title="Whatever">
    <img class="lightbox-thumbnail" src="images/filenameB-thumb.jpg">
</a>

<a href="images/filenameC.jpg" data-lightbox="lightbox-group-1" title="Whatever">
    <img class="lightbox-thumbnail" src="images/filenameC-thumb.jpg">
</a>

Click on these images below, and you will see that they are linked together in their lightboxes, with a back-arrow and forward-arrow. The group will 'cycle' if you continue clicking one arrow, starting at the beginning again (or starting at the end again, if the visitor is cycling backwards). There is no 'timed automatic' cycle, so your visitors can spend as long as they like looking at your images.
